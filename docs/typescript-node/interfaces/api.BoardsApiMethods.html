getBoards

  • getBoards(query?: { limit?: string; offset?: string; owner?: string; query?: string; sort?: "default" | "last_modified" | "last_opened" | "last_created" | "alphabetically"; teamId?: string }): Promise<{ body: BoardsPagedResponse; response: Response }>

  • Retrieves a list of boards that match the search criteria provided in the request. If you are an Enterprise customer and a Company Admin, you can retrieve all boards, including all private boards (boards that haven't been specifically shared with you) by enabling Content Admin permissions. To enable Content Admin permissions, see https://help.miro.com/hc/en-us/articles/360012777280-Content-Admin-permissions-for-Company-Admins. Note that you only get results instantaneously when you filter by team_id. If you use any other filter, you need to give a few seconds for the indexing of newly created boards before retrieving boards.

    Required scope

    <a target="blank" href="/reference/scopes">boards:read
